# PlayRanking
PlayRanking es una aplicación web para la gestión y comparación de rankings de juegos y jugadores. Está construida con una arquitectura de frontend y backend separados, utilizando tecnologías modernas como React para la interfaz de usuario, Express para el servidor y MongoDB para la base de datos.

Esta aplicación permite a los usuarios registrarse, iniciar sesión, y gestionar sus perfiles, además de crear y consultar rankings de sus juegos favoritos. El proyecto está diseñado para ser fácil de usar, escalable y con una estructura clara que facilita su mantenimiento y desarrollo futuro.

## Tecnologías utilizadas
- Node.js
- Express
- MongoDB
- React
